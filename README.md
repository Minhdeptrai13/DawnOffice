## 🌟 Tính Năng Nổi Bật (Key Features)

- 📄 **DawnDocument (Word)**: Trình soạn thảo văn bản chuẩn Tiptap WYSIWYG, hỗ trợ định dạng đa dạng, xuất bản tệp `.docx` và PDF.
- 📊 **DawnSheets (Excel)**: Bảng tính công thức thông minh (SUM, AVERAGE, COUNT...), định dạng ô, Pivot Table, sắp xếp & lọc dữ liệu.
- 🎨 **DawnSlides (PowerPoint)**: Trình tạo slide thuyết trình chuyên nghiệp, chế độ trình chiếu toàn màn hình (F5), phím tắt nhanh.
- 🔐 **Google Pro VIP Account**: Đăng nhập xác thực OAuth 2.0 an toàn, hiển thị huy hiệu viền Hoàng Gia mạ vàng (Golden Glowing Border).
- 💻 **Đa Nền Tảng (Windows & Linux)**: Đóng gói tệp cài đặt `.exe` / `.msi` cho Windows và `.AppImage` / `.deb` cho Linux.
- 🌙 **Giao Diện Thông Minh**: Hỗ trợ 4 chế độ Theme (Tự động theo OS, Chế độ Sáng, Tối Dark Mode, và Bảo Vệ Mắt Sepia).

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)

- **Frontend**: React 19, TypeScript, Vite, Vanilla CSS Design System, Lucide Icons.
- **Backend / Desktop Framework**: Rust, Tauri v2, Native OS File API.
- **Security & Cloud**: Google OAuth 2.0.

---

## 🚀 Hướng Dẫn Cài Đặt & Phát Triển (Development Setup)

```bash
# 1. Cài đặt các thư viện phụ thuộc (Install dependencies)
npm install

# 2. Chạy ứng dụng chế độ Development
npm run dev

# 3. Chạy ứng dụng Desktop Tauri
npm run tauri dev

# 4. Đóng gói bộ cài đặt sản phẩm (Windows .exe / .msi)
npm run tauri build
```
